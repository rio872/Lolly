# Lolly Jewellery Website

Lolly is a responsive jewellery frontend website built using HTML, CSS, and JavaScript.  
It includes a homepage, catalogue page, product sections, and image-based jewellery collections.

## Project Overview

This project is a static frontend website for a jewellery brand.  
The website is designed to showcase jewellery products such as rings, necklaces, earrings, and bracelets in a clean and attractive layout.

## Features

- Responsive homepage
- Jewellery catalogue page
- Product images with names and prices
- Organized product categories
- Navigation between pages
- Hover effects on product cards
- Clean and simple UI design
- Mobile-friendly layout
- Static frontend project, no backend required

## Technologies Used

- HTML5
- CSS3
- JavaScript
- VS Code Live Server
